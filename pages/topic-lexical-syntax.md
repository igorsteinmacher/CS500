# Topic: Lexical Syntax

**(Nov-30)**

In this topic, we will understand the details of the role of lexical and concrete/abstract syntax analysis in programming languages. Based on examples backed by the grammar of Clite we will explore the transformation from source code to lexical units to parse trees.
