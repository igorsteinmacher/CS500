# Topic: Declarative Languages -- Functional Programming

**(Nov-7)**

In this topic, we will introduce the main characteristics of the declarative paradigm, focusing on functional programming. We will follow an example-based approach to show the basics of a functional language. The students will have the opportunity to interact with basic examples written in Haskell.

Code Examples used in class provided

Slides provided:

- Functional Programming
- A hint of Haskell

Readings available:

- The answer to the question "What is a functional programming" is given by the Haskell documentation: <https://wiki.haskell.org/Introduction#What_is_functional_programming.3F>
- Hudak, Paul; Peterson, John; Fasel, Joseph. A gentle introduction to Haskell. Available: <https://www.haskell.org/tutorial/>
- Chapter 15 from Sebesta, Robert. *Concepts of Programming Languages **(15.1, 15.2, and 15.3 are MANDATORY -- Will be used for the quiz)***
- Chapter 11 from Gabbrielli, Maurizio, Martini, Simone. Programming Languages: Principles and Paradigms (Undergraduate Topics in Computer Science)
