# Module 6: Programming Languages: Grammar and Parse Trees


This module will cover how the compiler design basics, focusing on how the syntax analyzer works. To start, we will talk about grammar and how it is used to validate a language, going through the different representations (BNF, EBNF) and discussing ambiguity. Following, the focus will be on lexical syntax, and the conversion of tokens into lexemes and generating syntax parse trees. We wrap up by talking about names, variable binding and scoping
