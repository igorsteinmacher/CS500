# Slides and Readings: Programming Language Design Evaluation Criteria


In this topic we will walk through some of the aspects that can be considered when choosing a "good" programming language. The analysis will include simplicity, reliability, orthogonality, support, and costs and how each of them needs to be considered.

  Programming Language Evaluation Criteria Part 2: Writability, Reliability, Cost: <https://medium.com/@thisisfordeveloper/programming-language-evaluation-criteria-part-2-writability-reliability-cost-6c3029c9d957>
